## Visão geral

Biblioteca com os componentes em React para seguindo o desing do [Anibook](htpps://anibook.com.br).

### Requisitos

- [Node.js](https://nodejs.org/en/)

### Instalação

Instalação com Yarn ou Npm:

```bash
  yarn add anibook-ui
```

```bash
  npm install anibook-ui
```
